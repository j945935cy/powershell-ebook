PowerShell 學習教材

從零開始，慢慢學會用 PowerShell 自動處理工作

在 Windows 系統中，PowerShell 是最強大的自動化工具之一。 這本教材從最基礎的觀念出發，用白話文帶你一步步學會真正有用的指令。

開始閱讀 查看章節

適合誰？

  • PowerShell 完全新手
  • Windows 用戶想學自動化
  • 希望學了馬上能用的人
  • 希望不靠記憶體背語法的人

學習路徑

  1. 完成 認識 PowerShell 是什麼
  2. 完成 學會常用指令與路徑操作
  3. 完成 掌握變數、條件判斷、迴圈
  4. 完成 撰寫腳本並實際自動化
  5. 完成 了解進階方向

學完你會什麼

  • PowerShell 核心觀念與常用指令
  • 撰寫腳本自動整理檔案、批次操作
  • 查詢系統資訊並產生報告
  • 建立有參數的 .ps1 腳本
  • 繼續自學的方向與資源

章節目錄

  1. 01 PowerShell 是什麼？為什麼值得學？ 認識 PowerShell、與 cmd 的差別、實際用途
  2. 02 五分鐘認識核心：介面、指令、物件觀念 Cmdlet 命名規則、物件、管線、Get-Help
  3. 03 開始操作：開啟 PowerShell、執行基本指令 開啟方式、執行政策、第一個指令
  4. 04 常用指令入門：檔案、資料夾、路徑操作 Get-ChildItem、Set-Location、New-Item
  5. 05 常用指令實戰：查看、複製、搬移、刪除 Copy-Item、Move-Item、Remove-Item、Get-Content
  6. 06 變數、陣列、字串與基本資料處理 $變數、陣列、字串操作、型別
  7. 07 條件判斷與迴圈 if/else、foreach、while、switch
  8. 08 PowerShell 腳本入門 .ps1 檔案、執行政策、參數、常見問題
  9. 09 實用小自動化 整理檔案、批次改名、系統資訊查詢
  10. 10 持續練習慢慢變達人 學習策略、常見錯誤、進階方向
