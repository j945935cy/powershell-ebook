# powershell-ebook1

PowerShell 學習教材：五分鐘認識核心，三小時學會常用操作，持續練習慢慢變達人

## 專案說明
這是一個使用 HTML、CSS、JavaScript 製作的靜態教材網站，
可直接部署到 GitHub Pages。

## 技術
- HTML
- CSS
- JavaScript

## 本地開啟
直接用瀏覽器開啟 `index.html` 即可。

## GitHub Pages 部署方式
1. 將專案 push 到 GitHub
2. 到 repository 的 Settings
3. 點選 Pages
4. Source 選擇 `GitHub Actions` 或 branch
5. 儲存後即可發布

## 適合對象
- PowerShell 初學者
- Windows 使用者
- 想學命令列與自動化的人
